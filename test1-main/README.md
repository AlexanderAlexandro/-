# Currency Flow

Локальный запуск на macOS:

1. Откройте файл [start.command](/Users/a.vostretsov/Downloads/test1-main/start.command) двойным кликом
2. Откроется окно Terminal и автоматически запустится локальный сервер
3. Браузер откроет конвертер на `http://localhost:8000`

Альтернатива через терминал:

```bash
cd /Users/a.vostretsov/Downloads/test1-main
./start.command
```

Если захотите остановить сервер, вернитесь в окно терминала и нажмите `Ctrl+C`.

Публикация для открытия с телефона или любого компьютера:

Самый простой вариант без Git:

1. Откройте [Netlify Drop](https://app.netlify.com/drop)
2. Перетащите папку `/Users/a.vostretsov/Downloads/test1-main`
3. Получите публичную ссылку и открывайте её в обычном браузере с телефона
